Encryption & Decryption Web App

A simple web app to encrypt and decrypt messages using a secret key.

Project Files
main.py – Backend (Python Flask)
encrypt.html – Encrypt message page
decrypt.html – Decrypt message page
style.css – Styling
script.js – JavaScript functions
venu/ – Virtual environment (can be ignored)
main.cpython-313.pyc – Compiled file (auto-generated, ignore)

Features
Encrypt messages 
Decrypt messages 
Uses a secret key
Simple UI

How to Run
Install dependencies
pip install flask cryptography
Run the project
python main.py
Open in browser
http://127.0.0.1:5000/
